# schedule_app_develop_lsj_spring9_ch3
일정 관리 앱
* Author : LEE SEOJUN

## 프로젝트 소개
일정 관리 API를 구현<br/>
1. 일정을 생성, 조회, 수정, 삭제를 할 수 있습니다.
2. 유저를 생성, 조회, 수정, 삭제를 할 수 있습니다.
3. 비밀번호를 사용하여 회원가입을 할 수 있습니다.
4. 로그인 기능이 있습니다.
5. 로그아웃기능이 있습니다.
6. 조회와 회원가입, 로그인을 제외한 모든 기능은 로그인을 먼저 하셔야합니다.
